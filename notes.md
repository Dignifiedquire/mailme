# MailMe


## Features

* Fast (UI, Sync, Search)
* Simple and nice UI
* Multiple Accounts
* IMAP & Gmail
* Full offline sync

## Storage


* [maildir] format

```
~/Library/MailMe/
|
|- Email
|  |
|  `- _account01_
|     |
|     |- tmp
|     |- new
|     `- cur
|
`- Data
```
